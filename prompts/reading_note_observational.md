## 类型任务

本类型导读侧重**暴露/干预与结局的关联描述**、**偏倚风险**与**随访完整性**——**不**将观察性结果写成 RCT 级因果断言。

**MINORS 关注点映射**（内部思考，不输出打分表）：

| MINORS 条目 | 本 prompt 落点 |
|-------------|----------------|
| 1 明确研究目的 | 「研究问题」 |
| 2 连续纳入患者 | 「设计与人群」 |
| 3 前瞻性数据收集 | 「设计与人群」 |
| 4 终点与目的一致 | 「主要结果」 |
| 5 终点评估无偏倚（盲法） | 「设计与人群」「局限」 |
| 6 随访时长适当 | 「设计与人群」 |
| 7 失访 <5% 或说明 | 「设计与人群」「局限」 |
| 8 前瞻性样本量计算 | 「设计与人群」 |
| 9–12 对照组、同期、基线均衡、统计（仅比较性研究） | 「设计与人群」「暴露或干预」「局限」 |

**与其它类型的边界**：

- 有随机分配且主要目的是比较干预效应 → `reading_note_rct.md`
- 单臂病例系列无对照：仍用本模板，「暴露或干预」节写治疗方案，「局限」强调无对照与选择偏倚

**表述纪律（观察性特有）**：

- 用「关联」「差异」「与…相关」；避免「证明」「显著优于」除非文中原话且你标明为作者表述
- 回顾性研究因果推断弱——在「局限」或「临床启示」中一句话点明证据等级

## 输出模板

### 研究问题

- 临床问题一句话（可用 PICO 变体：**P** 人群、**E/I** 暴露或干预、**C** 对照或参考组若有、**O** 结局）
- 研究设计标签（从下列选一或组合，以文为准）：前瞻性队列 / 回顾性队列 / 病例对照 / 病例系列 / 前后对照 / 非随机比较
- 是否比较性研究（有/无对照组）

### 设计与人群

| 字段 | 内容 |
|------|------|
| 国家/地区与中心 | |
| 样本量 | 筛选→入组→完成分析（文中有则写） |
| 人群特征 | 年龄、性别、适应症、严重程度、 Fitzpatrick 分型等文中有则写 |
| 纳入方式 | 是否连续纳入；回顾性数据库/登记来源（文中有则写） |
| 分组/暴露定义 | 如何定义暴露组、对照组或无对照 |
| 数据收集 | 前瞻性方案 vs 回顾性图表回顾 |
| 随访时长与时间点 | |
| 失访/脱落 | 人数、比例；是否组间均衡 |
| 样本量估算 | 文中有则写，无则「文中未报告」 |
| 终点评估盲法 | 客观/主观终点是否盲法评估 |

### 暴露或干预

- **暴露因素或干预方案**：设备/产品/手术方式、参数、疗程、合并治疗
- **对照或参考组**（若有）：如何选取、是否为同期对照、基线是否可比
- **混杂因素**：文中是否记录或统计调整（年龄、性别、既往治疗、操作者等）

### 主要结果

**只写文中原报数据**。

- 主要结局指标定义（评估方法、时间点、量表名称与方向）
- 核心结果：效应量（OR/RR/HR/β 等）、调整变量、95%CI/p（文中有则写）
- 无对照的病例系列：报告改善率、均值变化、应答率等（文中原报）
- 关键图表结论（文字概括，不贴图）

可选表格（比较性研究且数据清晰时）：

| 结局 | 暴露/干预组 | 对照组 | 效应（文中原报） |
|------|-------------|--------|------------------|
| | | | |

### 安全性

- 并发症或不良事件（观察性研究常报告不全，如实写）
- 全文未报告 →「文中未报告不良事件」

### 局限

结合 MINORS 思维与文中 Discussion，覆盖（有则写，无则不必凑数）：

- 选择偏倚、信息偏倚、混杂、失访、单中心、无对照、测量偏倚（主观评估未盲法）
- 回顾性设计、随访过短、行业资助、结局指标替代性
- **证据等级一句话**（如「回顾性队列，因果推断弱于 RCT」）
- **不要**输出 MINORS 总分

### 临床启示

- 对临床实践的**参考点**（非推荐），2–4 条 bullets
- 与 RCT 证据相比的位置（若文中有讨论则转述；无则「文中未报告与 RCT 对照」）
- 外推 cautions：人群、操作者经验、设备代际是否与读者场景一致
