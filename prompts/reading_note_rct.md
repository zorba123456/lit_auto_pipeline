## 类型任务

本类型导读侧重**因果推断可信度**与**主要终点的可量化结果**。

**应覆盖**（映射 CASP）：

| CASP 区块 | 本 prompt 落点 |
|-----------|----------------|
| A · 基本设计是否有效（随机、分配隐藏、筛查三问） | 「设计与人群」 |
| B · 方法学是否可靠（盲法、基线、除干预外是否同等、随访/失访） | 「设计与人群」「干预与对照」 |
| C · 结果是什么（效应大小、精度 CI/p、各结局） | 「主要结果」 |
| 伤害与成本 | 「安全性」 |
| D · 本地是否适用（人群差异、重要结局是否齐全、利弊权衡） | 「局限」「临床启示」 |

**与其它类型的边界**：

- 非随机、无对照或回顾性比较 → 用 `reading_note_observational.md`
- 系统综述/Meta → P2+ `reading_note_review.md`
- 动物/细胞/机制为主、无临床入组 → P2+ `reading_note_basic.md`

**医美常见注意点**：

- 干预节须写清：设备/产品名称、能量/剂量/密度、治疗次数与间隔、麻醉、术后护理、是否允许合并治疗
- 主观终点（医师评估、患者满意度）须注明评估者是否盲法、量表方向（越高越好/越低越好）
- 行业资助、单中心、短期随访、替代终点 → 在「局限」中点出（文中有则写）

## 输出模板

按下列标题逐项填写。每节 1 段或 bullets 均可，以信息密度为准；**主要结果**若有 ≥2 个并列终点，优先用表格。

### 研究问题

- 用 **PICO 一句话**：**P** 人群与适应症、**I** 实验干预、**C** 对照、**O** 主要结局（文中原称）。
- 标明研究设计（如「单中心、双盲、平行组 RCT」「三臂 RCT」）。
- 若摘要与正文对主要终点表述不一致，以 Methods 为准并注明。

### 设计与人群

- **入组**：纳入/排除要点、招募途径、中心数与国家/地区
- **随机化**：方法、分配隐藏（allocation concealment）；文中未报告 →「文中未报告」
- **盲法**：受试者 / 操作者 / 结局评估者各自是否盲法；无法盲法的理由（医美设备试验常见）
- **样本量**：计划入组 vs 实际分析例数（ITT 或 PP，文中有则写）；是否报告样本量估算与把握度
- **基线**：组间主要特征是否均衡（年龄、性别、问题严重程度、既往治疗等）
- **随访**：随访时长、时间点；失访/脱落人数与比例；组间是否同等随访

### 干预与对照

- **实验组**：干预名称；设备/产品/药物；关键参数（能量 J/cm²、脉宽、针深、单次量、疗程）；操作者资质要求；是否允许合并治疗
- **对照组**：安慰、空白、假设备、标准治疗或阳性对照的具体方案（与实验组可比的疗程）
- **除研究干预外**：组间处理是否一致（术后护理、镇痛、防晒等）

### 主要结果

**只写事实**（文中原报数据），不写「意义重大」等评价性语句。

- **主要终点**（文中原称）：各组结果、组间差异、效应量（MD/SMD/RR/OR/HR 等，文中有则写）
- **精度**：95%CI 或 p 值（文中有则写）；未报告 →「文中未报告」
- **次要终点**：仅列与临床决策相关的 1–3 项（其余可省略）
- 若使用验证量表（GAIS、ECCA、MASI、WSRS、VAS、FWS 等）：写明评分方向、基线与随访变化
- 亚组或敏感性分析：仅在有临床解读价值时简述

可选表格（当文中有清晰组间数据时）：

| 结局 | 实验组 | 对照组 | 效应（文中原报） |
|------|--------|--------|------------------|
| [主要终点] | | | |

### 安全性

- 不良事件（AE）类型、发生率、严重程度分级（文中原报）
- 严重不良事件（SAE）或导致退出治疗的事件
- 作者对 AE 与干预关联性的判断方式（文中如何表述）
- 全文未报告 →「文中未报告不良事件」

### 局限

- **作者自述局限**（转述 Discussion/Limitations 要点，2–4 条）
- **读者视角补充**（仅基于文中可观察到的缺陷，无则省略）：如单中心、短期随访、行业资助、替代终点、高失访、外推性受限、操作者经验依赖
- **不要**输出 MINORS/CASP 打分

### 临床启示

**含意与 cautions**，非治疗建议。

- 对**同类医美临床场景**的可借鉴点（2–4 条 bullets，克制表述）
- **外推 cautions**：试验人群、设备代际、联合治疗、操作者水平是否与读者场景一致
- **证据强度一句话**（如「单中心小样本 RCT，主要终点阳性但长期安全性未明」）
- 区分「文中作者结论」与「读者需谨慎之处」
