# AES 中文表述规范

> **真源文件**：[`data/style_guide.json`](../data/style_guide.json)  
> **维护**：8300 侧边栏「文档维护」→ [中文表述规范](http://localhost:8300/projects/aes-writing-style.html)（**不单独开端口**，保存即写回 JSON）  
> **调用**：`prompts/style_guide_loader.py` 读 JSON → 注入导读/译题 prompt（下次 worker 运行自动生效）

适用于：**标题翻译**、**中间页 AI 导读**、**摘要直译**及后续全文翻译链。无人工逐条审校时，靠本规范约束 LLM 输出，降低机翻感与术语漂移。

---

## 1. 用词：避免 AI 滥用，灵活换说

| 慎用 / 限用 | 可替换为 | 说明 |
|-------------|----------|------|
| 核心 | 主要、关键、要点、重点、首要 | 勿段段「核心」 |
| 至关重要、极其重要 | 十分重要、需重点关注、影响较大 | 降调强调语 |
| 虽然 / 尽管 | 二者互换 | 同篇勿机械重复 |
| 创新性、颠覆性 | 改良点、新做法、与既往的区别 | 导读写「作者认为…」 |
| 显著、明显 | 附 P 值/CI 的具体表述 | 有数据则写数字 |
| 展示了、揭示了 | 显示、表明、提示、结果为 | 减套话 |

## 2. 术语（固定译法）

| 避免 | 采用 | 英文 |
|------|------|------|
| 玻尿酸 | **透明质酸** | hyaluronic acid |
| 肉毒素、肉毒 | **肉毒毒素** | botulinum toxin |
| — | 富血小板血浆（PRP） | platelet-rich plasma |
| 填充物（产品语境） | **填充剂** | filler |
| 副作用（文献语境） | **不良事件** | adverse event |

首次出现：**中文（English）**；商品名、设备型号保留原文。

---

## 3. 句法组织

**减少机翻感**

- 长句拆短、短句合并；语序、主被动不必照搬英文。
- 少「被…所…」「进行了…的研究」。
- 同段避免重复同一主语开头。

**表意明确**

- 生僻概念换易懂说法，但不改医学含义。
- 指代清楚（「该组」对应哪一组）。
- 数字、单位、随访时间与原文一致。

**格式**

- 人名、机构、产品、公司不译。
- 数字与中文之间不空格（如 3 个月、50%）。

**语气（导读）**

- 平实陈述；无营销腔、无「你应该/推荐采用」。

## 5. 维护方式

1. 浏览器打开 **8300** → 侧边栏「文档维护」→ [中文表述规范](http://localhost:8300/projects/aes-writing-style.html)
2. 增删改词条 → **保存到仓库**（自动备份至 `data/backups/style_guide/`）
3. 无需重启服务：`note_worker` / `doubao_rpa` 等每次生成时 `style_guide_loader` 读磁盘最新 JSON
